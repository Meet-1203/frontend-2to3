What is protocol? A protocol in HTML is a standard that defines hoe data is transmitted between computers. In HTML, the protocol property sets or returns the protocol part of the href attribute value,which specifies the destination of a link in an area. The protocol property can also set or return the protocol of the current URL,including the colon(:). Hypertext Transfer Protocol(HTTP) is a protocol that is used to fetch HTML documents and other resources on the web. HTTP is a client-server protocol ,which means that the recipient ,usually a web browser,initiates requests. HTTP messages are exchanged between clients and servers,with requests sent by the client and responses sent by the server.
